# Leptos Web UI 视觉回归检查清单

在改动 `frontend-leptos/styles/**` 或 `frontend-leptos/src/lib.rs` 中影响布局/主题的视图后，发版或合并前建议逐项手测（无需自动化截图）。

## 主题

- [ ] **深色**：顶栏、对话区、侧栏、状态栏层次清晰；`muted` 文案可读。
- [ ] **浅色**：同上；`border-subtle` 与背景仍可分辨。

## 视口

- [ ] **宽屏（>720px）**：顶栏「工作区 / 任务 / 状态 / 主题」全部可见；无多余换行。
- [ ] **窄屏（≤720px）**：次要操作收入「更多」菜单；点击遮罩可关闭；菜单内切换侧栏/主题正常。

## 聊天与侧栏

- [ ] **消息元信息**：每条气泡上方有角色标签；新消息显示 `HH:MM`（旧本地数据可为空）。
- [ ] **长消息 / 流式**：气泡宽度与滚动正常；加载中 typing 点仍可见。
- [ ] **工作区卡片头**：显示「N 项」或「加载中…」/「错误」。
- [ ] **任务卡片头**：显示「完成数/总数」或加载/错误态。
- [ ] **侧栏显隐**：宽度过渡平滑；折叠后主区无双边框。

## 审批条

- [ ] **默认折叠**：仅见标题行 + 命令预览截断。
- [ ] **展开**：完整 `pre` 与三个按钮可用。

## 会话弹窗

- [ ] **键盘焦点**：`Tab` 到会话行时可见 `focus-visible` 环。
- [ ] **当前会话**：左侧色条与背景区分明显。

## 骨架与状态栏

- [ ] **骨架屏 → 内容**：工作区/任务/状态加载完成后无布局跳动。
- [ ] **角色下拉**：外观与状态 chip 协调；`focus-visible` 有环。
- [ ] **`prefers-reduced-motion: reduce`**：列表渐入、状态点脉冲、侧栏过渡、审批条展开动画关闭或等价静态呈现。

## 构建

- [ ] `cd frontend-leptos && trunk build` 成功（确认 `@import` 子样式被打包）。
